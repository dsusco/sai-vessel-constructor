# Steam and Iron Vessel Constructor

A vessel constructor for Steam and Iron.

## The TL;DR Start

    $ git clone git@github.com:dsusco/sai-vessel-constructor.git
    $ cd sai-vessel-constructor/
    $ npm ci
    $ npm run dev
